รูปตนเอง

เหตุผลที่อยากเรียน Full Stack Web Development

ผมสนใจ Full Stack Web Development เพราะต้องการเรียนรู้การพัฒนาเว็บไซต์ตั้งแต่ฝั่งหน้า (frontend) จนถึงฝั่งหลัง (backend) เพื่อที่จะสามารถสร้างโปรเจกต์จริงที่ใช้งานได้ครบวงจร และเพิ่มทักษะที่นำไปใช้ในสายอาชีพได้

ความคาดหวังต่อวิชานี้

ผมคาดหวังว่าจะได้ฝึกทักษะการทำงานเป็นทีม เพิ่มความสามารถในการแก้ปัญหาเชิงโปรแกรม และเรียนรู้แนวทางการพัฒนา ระบบเว็บที่สมบูรณ์ทั้งในเชิงเทคนิคและการออกแบบประสบการณ์ผู้ใช้

จุดอ่อนทางการเขียนโปรแกรม และแนวทางการพัฒนา

จุดอ่อน: ชอบลืมและชอบพึ่งai มากไป

แนวทางพัฒนา: ฝึกแก้ปัญหาบนและlogic การคิดและเรียนรู้ฝึกฝนให้มาก

เว็บไซต์ที่ชื่นชอบ / แหล่งแรงบันดาลใจ

  • GitHub — แหล่งค้นหาโปรเจกต์ตัวอย่าง และโค้ดจากนักพัฒนาทั่วโลก
  • Chatgpt — ไว้หาความรู้เพิ่มเติม
  • Youtube — คลายเครียดฟังเพลงชิลๆ
